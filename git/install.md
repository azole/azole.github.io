# 安裝 git


確認電腦裡是否有安裝過 git:

執行以下指令，如果有正確出現版本號，就是安裝過 git

```bash
git --version
```



沒有安裝過的到 [https://git-scm.com/download/win](https://git-scm.com/download/win) 下載安裝檔
